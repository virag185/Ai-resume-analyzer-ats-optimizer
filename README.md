# 🚀 AI Resume Analyzer & ATS Optimizer

An AI-powered web application that analyzes resumes, calculates ATS compatibility scores, and provides intelligent suggestions to improve hiring success.

---

## ✨ Features

* 📄 Resume parsing (PDF/DOCX)
* 📊 ATS score calculation
* 🔍 Keyword matching with job descriptions
* ✨ AI-powered bullet point enhancement
* 📈 Smart suggestions dashboard

---

## 🛠 Tech Stack

* Frontend: Next.js, Tailwind CSS
* Backend: Node.js / FastAPI
* AI: OpenAI API

---

## 🚀 Project Structure

```
client/   → Frontend (Next.js)
server/   → Backend API
```
